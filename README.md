# 🔐 Log Analyzer (Mini SIEM)

## 📌 Description
Analyzes log files and detects suspicious IP activity.

## 🚀 Features
- Detect repeated IP access
- Threshold-based alerts
- JSON output

## 📋 Requirements
- Python 3.7+

## ▶️ Run
```bash
python analyzer.py
```

## 📊 Output
Generates `output.json` com IPs suspeitos e seus timestamps.
